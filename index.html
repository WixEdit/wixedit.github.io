About WixEdit

WiXEdit is an editor for XML source files for the Windows Installer XML (WiX) toolset to build MSI and MSM setup packages. WixEdit tries to make the editing of WiX XML source files simple and comprehensible for everyone. Without any or little knowledge of WiX or even XML, you can create MSI installer packages for your software! WixEdit is the only editor for WiX XML source files which is activly being developed. If you like what you see please help out and join us at github!

Latest version

This version of WixEdit contains a new wizard to help you create new msi files from scratch. The wizard helps you to create a working installer with files and functionality like shortcuts, virtual directories, user interface and more in a few clicks! Please let me know what templates you wish to see in the wizard, or better provide small samples of which templates can be made!

Goals

The main goal is to help create MSI installers using the WiX toolset. This project helps to edit WiX source files, and gives you a better overview over your installer. It will still be necessary to have some knowledge about the WiX toolset, but the program helps to display the files in a structured manner and maintain an better overview while navigating through the wxs file. But for simple installations, the wizard will guide you till a working installation.
